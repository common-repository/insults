Generate random insults on your WordPress website using shortcode.

== Description ==

**Insults** is an extremely simple plugin that'll generate a random insult on your website every hour. 

Features include:

* Customize the html that wraps around the insult on your page.
* Data is cached locally (using the WordPress transient API) for 60 minutes.
* Over 30,000 insults.

Related links:

* [Usage](http://www.beliefmedia.com/wp-plugins/insults.php)
* [BeliefMedia](http://www.beliefmedia.com/)

== Installation ==

**To install the plugin manually:**

1. Extract the contents of the archive (zip file)
2. Upload the insults folder to your '/wp-content/plugins' folder
3. Activate the plugin through the Plugins section in your WordPress admin panel.

**Upload via the WordPress administration panel:**

1. Click on "Plugins" in the left panel, then click on "Add new".
2. You should now see the Install Plugins page. Click on "Upload".
3. Click on Browse and select your "insults.zip" file.
4. Click on "Install now", activate it and you're done!

== Frequently Asked Questions ==

**What will the plugin do?**

The Insults plugin will retrieve a random insult from our API and display the text anywhere on your website.

**I can't see shortcode in my WordPress sidebar. What should I do?**

There's an option to uncomment a single line at the bottom of the insults.php file that'll enable shortcodes in widgets. Additional information is available on our support website.
